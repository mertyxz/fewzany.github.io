ชื่อทางการ : คอร์เดีย

ชื่อทางวิทยาศาสตร์ : Geiger tree, Geranium tree

สกุล : Cordia

วงศ์ : Boraginaceae

กลุ่มพรรณไม้ :ไม้ยืนต้น/ไม้พุ่ม

ลักษณะทางพฤกษศาสตร์ :
คอร์เดียเป็นไม้ยืนต้นหรือไม้พุ่มที่มีความสูงประมาณ 5-15 เมตร ขึ้นอยู่กับชนิดพันธุ์
ใบมีลักษณะเรียบง่าย รูปไข่หรือรูปวงรี ขอบใบเรียบหรือหยักเล็กน้อย ใบมีสีเขียวสด
ดอกของคอร์เดียมีขนาดเล็กถึงกลาง โดยมีสีขาว ส้ม หรือแดง ขึ้นอยู่กับชนิดพันธุ์ ดอกมีลักษณะเป็นช่อหรือดอกเดี่ยว
ผลของคอร์เดียมีลักษณะเป็นผลกลมเล็ก เมื่อสุกมีสีเหลืองถึงดำ มักมีกลิ่นไม่พึงประสงค์ในบางสายพันธุ์

สภาพนิเวศ : คอร์เดียเจริญเติบโตได้ดีในพื้นที่ที่มีแสงแดดจัด ดินควรมีการระบายน้ำดี และสามารถทนต่อความแห้งแล้งได้ดี

สภาพนิเวศวิทยา : คอร์เดียเป็นพืชที่ปรับตัวได้ดีในพื้นที่แห้งแล้ง และสามารถเจริญเติบโตได้ในดินที่ไม่สมบูรณ์

ถิ่นกำเนิด : คอร์เดียพบได้ทั่วไปในภูมิภาคเขตร้อนและกึ่งเขตร้อน เช่น เอเชียใต้ เอเชียตะวันออกเฉียงใต้ และแอฟริกา

การกระจายพันธุ์ : พบได้ทั่วไปในภูมิภาคเขตร้อนทั่วโลก โดยเฉพาะในพื้นที่ที่มีสภาพอากาศร้อนและแห้ง

การปลูกและขยายพันธุ์ :
ขยายพันธุ์ได้โดยการเพาะเมล็ดและการตอนกิ่ง
การปลูกควรปลูกในดินที่มีการระบายน้ำดี และในพื้นที่ที่มีแสงแดดเต็มวัน

แหล่งอ้างอิง :ข้อมูลเกี่ยวกับคอร์เดียสามารถพบได้ในตำราเกี่ยวกับพรรณไม้และงานวิจัยด้านพฤกษศาสตร์
